# LampaTrakt

Форк плагина [Trakt.tv для Lampa](https://lampame.github.io/main/trakttv.js) от [LampaME](https://lampame.github.io/main/). Оригинальный плагин создан командой LampaME — все авторские права на исходный код принадлежат им. Этот форк создан с уважением к оригинальной работе и доработан для удобной работы с историей просмотров, статусами сериалов, несколькими аккаунтами и совместным просмотром.

Дополнения и изменения, внесённые в этом форке, распространяются под лицензией [MIT](LICENSE).

Текущая версия: **3.2.23**

## Возможности

### Авторизация
Вход через OAuth 2.0 с собственными Client ID и Client Secret из приложения Trakt. После авторизации токены хранятся локально. Поддерживается до 6 аккаунтов одновременно.

### Источник контента
Плагин регистрирует источник `trakttv`, который добавляет на главную страницу строки с подборками:
- **Трендовые** — самые обсуждаемые прямо сейчас
- **Ожидаемые** — самые антиципируемые новинки
- **Популярные** — лидеры по просмотрам за всё время
- **Рекомендации** — персональные рекомендации на основе вашей истории

Все карточки показываются с русскими названиями и постерами через TMDB. Разделы по жанрам (например, «Тренды: Боевик») отображаются на русском.

Поиск через источник TraktTV возвращает результаты на русском языке.

### Меню Trakt
Раздел в боковом меню Lampa:

| Раздел | Описание |
|---|---|
| Смотреть дальше | Следующие непросмотренные серии из вашего списка «Хочу посмотреть» |
| Хочу посмотреть | Все добавленные фильмы и сериалы |
| Мои сериалы | Сериалы с прогрессом: вкладки Активные / Завершённые / Брошенные / Все |
| История просмотров | Хронологическая лента с вкладками Сегодня / Вчера / Неделя / Месяц / Год |
| Избранное | Коллекция сохранённых фильмов и сериалов с фильтрами и сортировкой |
| Расписание | Ближайшие эпизоды и цифровые релизы фильмов |
| Рекомендации | Персональные рекомендации Trakt |
| Мои списки | Управление пользовательскими списками |
| Понравившиеся списки | Списки, добавленные в избранное |

Карточки во всех списках показываются с русскими названиями и постерами (через TMDB). В списках с сортировкой («Хочу посмотреть», «Избранное», пользовательские списки) доступны фильтры (Тип / Год / Жанр / Страна), выбор поля и направления сортировки, а также пункт **«Открыть случайный»** — открывает случайный элемент из текущей выборки (ожидаемые исключаются). Сортировка по умолчанию задаётся в настройках и применяется ко всем спискам и строке «Хочу посмотреть» на главной.

### Несколько аккаунтов

Плагин поддерживает до 6 аккаунтов Trakt одновременно. В верхней панели Lampa появляется иконка с номером активного аккаунта.

- Нажмите иконку → появится меню со списком аккаунтов
- Выберите аккаунт — плагин переключится и обновит все данные
- Для каждого аккаунта можно задать псевдоним (отображаемое имя)
- Можно показывать аватар профиля Trakt вместо номера слота
- Показывается статус Trakt VIP
- Если ни один аккаунт не авторизован, иконка показывает красный крестик, а клик по ней открывает настройки плагина для входа

**Тумблер отслеживания в шапке.** В меню значка есть пункт «Включить отслеживание просмотра» (Да/Нет) — то же, что в настройках. При одном авторизованном аккаунте меню значка содержит только этот пункт. Когда отслеживание выключено, значок Trakt в шапке становится белым; когда включено — остаётся фирменным фиолетовым.

**Гостевой вход.** Любой слот можно авторизовать как гостя на 24 часа — по истечении срока слот автоматически выходит из аккаунта.

### Совместный просмотр (Multiwatch)

Режим, при котором все списки фильтруются по **пересечению** списков «Хочу посмотреть» выбранных аккаунтов: показываются только те фильмы и сериалы, которые добавлены у **всех** участников.

**Как включить:**
1. Нажмите иконку аккаунта в верхней панели
2. Выберите «Совместный просмотр»
3. Отметьте аккаунты-участники и нажмите «Готово»

**Что фильтруется:** «Хочу посмотреть», «Смотреть дальше», История, Мои сериалы, Избранное, Расписание.

При включении на иконке появляется значок с двумя фигурами людей. Для выключения — выберите «Выключить» или переключитесь на одиночный аккаунт.

### Бейджи на постерах

На карточках фильмов и сериалов отображаются бейджи с логотипом Trakt:

- **Галочка** — фильм просмотрен или сериал завершён полностью
- **Полукруг** — сериал в процессе просмотра (есть прогресс, но не завершён и не брошен)
- **Закладка** — фильм или сериал в списке «Хочу посмотреть»
- **Дата цифрового релиза** — относительная дата («Завтра», «Через 2 дня», «12 июл») или «Скоро» для фильмов с предстоящим цифровым релизом
- **Шкала прогресса** (выключена по умолчанию, выбор — фильмы / сериалы / оба / нет) — полоска сверху постера с процентом просмотра

Видимость каждого бейджа настраивается отдельно.

### Строка «Хочу посмотреть» на главной

Список «Хочу посмотреть» на главном экране показывает только уже доступные фильмы и сериалы:
- Скрываются позиции с датой релиза в будущем
- Скрываются фильмы без известной даты выхода
- Скрываются фильмы с предстоящим цифровым релизом (они отображаются в «Расписании»)

### Реалтайм-обновление главной

После отметки серии/фильма как просмотренного строки на главной перестраиваются на месте, без полной перезагрузки страницы:
- **«Смотреть дальше»** — обновляется после отметки, паузы (добавление в прогресс) и завершения просмотра
- **«Хочу посмотреть»** — просмотренный фильм автоматически исчезает из строки при возврате на главную

Статус на странице описания (`S.. · E..` / «Просмотрено») также перерисовывается на месте.

### История просмотров
- Хронологический список всего просмотренного с вкладками по периодам (Сегодня, Вчера, Неделя, Месяц, Год)
- На постерах отображается дата просмотра и номер серии (S·E) для эпизодов
- Клик по карточке открывает полную информацию о фильме или сериале
- Долгое нажатие → удаление записи из истории на Trakt

### Мои сериалы

Хаб для отслеживания прогресса по сериалам с четырьмя вкладками:
- **Активные** — сериалы в процессе просмотра с прогресс-баром
- **Завершённые** — все просмотренные сериалы
- **Брошенные** — сериалы, добавленные в скрытые (dropped)
- **Все** — полный список

На карточках активных сериалов показывается кружок с остатком непросмотренных серий. В контекстном меню — пункты «Бросить» и «Возобновить».

### Статус просмотра

На карточке сериала или фильма кликабельный статус показывает текущее состояние и дату последнего просмотра:
- **Просмотрено полностью** — все вышедшие серии отмечены просмотренными
- **Смотрю · S2 · E5** — есть прогресс, показан последний просмотренный эпизод
- **Просмотрено / Не просмотрено** — для фильмов

Рядом со статусом «Просмотрено» в скобках указывается дата (например, `15 мая 2025`).

При нажатии на статус открывается меню:

**Для сериалов:**

| Пункт | Действие |
|---|---|
| Отметить как просмотрено полностью | Отмечает все сезоны и серии просмотренными |
| Отметить как не просмотрено | Удаляет весь сериал из истории |
| Хочу посмотреть / Не буду смотреть | Добавляет или убирает из «Хочу посмотреть» |

**Для фильмов:**

| Пункт | Действие |
|---|---|
| Отметить как просмотрено сейчас | Записывает просмотр с текущей датой |
| Отметить как просмотрено с неизвестной датой | Записывает просмотр без даты |
| Отметить как не просмотрено | Удаляет фильм из истории |
| Хочу посмотреть / Не буду смотреть | Добавляет или убирает из «Хочу посмотреть» |

### Управление историей из карточки
Кнопка «Добавить в историю» на странице фильма или сериала позволяет:
- Отметить весь сериал просмотренным
- Отметить последний сезон
- Отметить последний эпизод
- Убрать из истории

### «Хочу посмотреть» и коллекция
Кнопки добавления/удаления из «Хочу посмотреть» и коллекции Trakt прямо на странице фильма или сериала.

### Расписание
Раздел объединяет два типа контента:
- **Сериалы** — ближайшие эпизоды из вашего списка «Хочу посмотреть» (следующие 14 дней, по одной записи на сериал)
- **Фильмы** — фильмы из «Хочу посмотреть» с предстоящей датой цифрового релиза

Для эпизодов отображается скриншот кадра, для фильмов — постер или кадр из фильма.

На странице фильма строка «Цифровой релиз фильма» показывает полную дату `ДД.ММ.ГГГГ`, если релиз известен, или «Скоро» — для фильмов, у которых был недавний театральный релиз, но дата цифрового ещё неизвестна (те же, что получают бейдж «СКОРО» на постере).

### Авто-торрент

Кнопка «Авто-торрент» в меню «Смотреть» автоматически выбирает лучший торрент и запускает воспроизведение без лишних нажатий.

- **Для сериалов** — определяет следующий непросмотренный эпизод через Trakt API и ищет для него торрент
- **Для фильмов** — ищет торрент с учётом настроек озвучки и качества
- **Авто-торрент (случайная серия)** — отдельная кнопка только для сериалов: выбирает случайный сезон и эпизод и запускает поиск торрента
- **Прямо из «Смотреть дальше»** — если включить опцию «Авто-торрент из «Смотреть дальше»», выбор карточки в этой строке на главной сразу запускает Авто-торрент, минуя открытие карточки (для сериала — следующий непросмотренный эпизод)
- **Приоритет ранее открытых** — торренты, которые уже открывались в Lampa, проверяются первыми
- **Перебор торрентов** — если торрент не открылся, пустой или в нём нет нужной серии, плагин автоматически пробует следующий подходящий (до 30 с на попытку)
- **Отмена в любой момент** — кнопка «Назад» на пульте или клик по кнопке «Прервать автозапуск» на экране ожидания прерывают процесс и возвращают к карточке

Настройки: приоритетное качество (4K / 1080p / 720p / максимум), озвучка (дубляж / любая русская / без фильтра — раздельно для фильмов и сериалов), приоритет сортировки (качество или популярность), число попыток (все подходящие / до 5 / до 3).

Перед открытием списка торрентов ручной фильтр Lampa временно сбрасывается (чтобы автовыбор видел все качества) и восстанавливается после выбора.

По умолчанию выключена. Включается в настройках плагина.

### Синхронизация TorrServer ↔ Trakt

Двусторонняя тихая синхронизация прогресса просмотра:

- **TorrServer → Trakt**: при остановке до порога прогресс сохраняется через `POST /scrobble/pause`
- **Trakt → TorrServer**: при запуске Lampa или открытии карточки сериала прогресс из `/sync/playback` восстанавливается в `Lampa.Timeline`

Работает только для эпизодов, которые хотя бы раз воспроизводились через TorrServer.

### Рекомендации
Персональные рекомендации Trakt. В контекстном меню карточки (и в хабе рекомендаций) доступен пункт **«Скрыть рекомендацию»** — убирает элемент из рекомендаций Trakt (`DELETE /recommendations`). В настройках можно скрывать уже просмотренное.

### Контекстное меню
Пункт «Списки Trakt» в контекстном меню любой карточки — быстрое добавление в пользовательский список.

### Мои списки
- Создание, переименование, изменение описания и удаление собственных списков
- Добавление и удаление элементов внутри списков

## Установка

1. В Lampa откройте **Настройки → Плагины → Добавить плагин вручную**
2. Вставьте URL файла `trakttv.js`
3. Перезагрузите Lampa
4. Откройте **Настройки → Trakt.TV** и укажите Client ID и Client Secret, затем нажмите **Войти**

### Получение Client ID и Client Secret

1. Перейдите на [trakt.tv/oauth/applications](https://trakt.tv/oauth/applications) → **New Application**
2. Name: любое (например, Lampa)
3. Redirect URI: `urn:ietf:wg:oauth:2.0:oob`
4. Permissions: отметьте `/checkin` и `/scrobble`
5. Нажмите **Save App**
6. Скопируйте Client ID и Client Secret в настройки плагина

Или нажмите кнопку **«Как получить Client ID и Secret?»** прямо в настройках плагина.

## Цвета оформления

Плагин использует настоящий фирменный фиолетовый Trakt — `#9F42C6` (цвет иконки официального iOS-приложения, он же первый стоп логотип-градиента). Это визуально отделяет элементы плагина от нативных интерфейсов Lampa.

### Фирменные цвета Trakt

| Цвет | Где используется | Почему |
|---|---|---|
| Сплошной `#9F42C6` | Везде по бренду: значок Trakt в шапке, цифра бейджа аккаунта, бейджи на постерах (просмотрено / хочу посмотреть / история / цифровой релиз), полоска-разделитель в списках, подчёркивание активных вкладок и сортировок, текст статуса «Смотрю», ссылки | Настоящий цвет иконки Trakt; не ломает логику значка (белый при выключенном отслеживании, инверсия при фокусе) |
| Градиент `#9f42c6 → #b4339a → #cf2061 → #f50613` | Только заливка прогресс-бара | Единственный градиент в плагине |
| Полупрозрачный `rgba(159,66,197, …)` | Фон и подсветка выделения активных вкладок и кнопок сортировки, бейдж «Смотрю» (двухтоновый полукруг) | Лёгкий брендовый акцент |

### Функциональные цвета (статусные, не бренд)

| Цвет | Где | Почему |
|---|---|---|
| Зелёный | Индикатор «ok», бейдж Trakt VIP, кнопка «Готово» | Успех / активно |
| Красный | Ошибка значка, крестик «нет авторизованных аккаунтов» | Ошибка / внимание |
| Сине-зелёный градиент | Фон карточки «Создать список» | Нейтральный, намеренно не брендовый |
| Белый / чёрный с прозрачностью | Текст, оверлей авто-торрента, разделители, фон фокуса, тени | Нейтральная база интерфейса |
| Серые | Trakt VIP «выключен», второстепенный текст, фоны | Низкий акцент |

## Технические детали

- **Trakt API v2** — авторизация OAuth 2.0, все запросы через `Bearer` токен
- **TMDB** — локализация названий и постеров на языке интерфейса Lampa
- **Мульти-аккаунт** — до 6 слотов в `Lampa.Storage` (`trakt_accounts`); активный слот — `trakt_active_slot`
- **Scrobbling** — основной путь: `Lampa.Timeline 'update'` → `processTimelineUpdate` при закрытии плеера; резервный: `Player 'destroy'` + 500 мс задержка
- **Внешний плеер** — слушатель `Player 'external'` (infuse, tvOS Pro, MX и др.): встроенный плеер не запускается, отметка происходит по финальному `Timeline 'update'` при возврате в Lampa
- **`_isPlayerActive` guard** — защита от ложных отметок при просмотре списков файлов торрента
- **Реалтайм-главная** — оптимистичные бейджи + live-rebuild строк «Смотреть дальше» / «Хочу посмотреть» без полной перезагрузки (настройка `trakt_realtime_upnext`)
- **Кэш бейджей** — `_watchedCache { movies, shows, completedShows, watchingShows, moviesWatchedAt }` из API параллельно
- **`hashMetaCache`** — `Map<hash, {card, season, episode}>` TTL 7 дней; связывает файл торрента с эпизодом
- **Кэш предстоящих фильмов** — `trakt_upcoming_movie_ids` с TMDB ID фильмов с будущим цифровым релизом; `trakt_soon_movie_ids` — фильмы с недавним театральным релизом без даты цифрового («Скоро»)
- Поддерживает Lampa 3.0+ (`Lampa.Maker`) и более старые версии (`InteractionCategory`)
