# LampaTrakt

Форк плагина [Trakt.tv для Lampa](https://lampame.github.io/main/trakttv.js) от [LampaME](https://lampame.github.io/main/). Доработан для удобной работы с историей просмотров, статусами сериалов, несколькими аккаунтами и совместным просмотром.

Текущая версия: **2.1.0**

## Возможности

### Авторизация
Вход через OAuth 2.0 с собственными Client ID и Client Secret из приложения Trakt. После авторизации токены хранятся локально.

### Источник контента
Плагин регистрирует источник `trakttv`, который добавляет на главную страницу строки с подборками:
- **Трендовые** — самые обсуждаемые прямо сейчас
- **Ожидаемые** — самые антиципируемые новинки
- **Популярные** — лидеры по просмотрам за всё время
- **Рекомендации** — персональные рекомендации на основе вашей истории

### Меню Trakt
Раздел в боковом меню Lampa с разделами:

| Раздел | Описание |
|---|---|
| Начать просмотр (Up Next) | Следующие непросмотренные серии сериалов из вашего watchlist |
| Список просмотра (Watchlist) | Все добавленные фильмы и сериалы |
| История просмотров | Хронологическая лента просмотренного контента |
| Расписание | Календарь выхода новых серий |
| Рекомендации | Персональные рекомендации Trakt |
| Мои списки | Управление пользовательскими списками |
| Понравившиеся списки | Списки, добавленные в избранное |

### Несколько аккаунтов

Плагин поддерживает одновременное хранение нескольких аккаунтов Trakt. Для быстрого переключения между ними в верхней панели Lampa появляется иконка с бейджем-номером активного аккаунта.

- Нажмите иконку → появится меню со списком аккаунтов
- Выберите аккаунт — плагин переключится и обновит все данные

### Совместный просмотр (Multiwatch)

Режим, при котором все списки фильтруются по **пересечению** watchlist-ов выбранных аккаунтов: показываются только те фильмы и сериалы, которые добавлены у **всех** участников.

**Как включить:**
1. Нажмите иконку аккаунта в верхней панели
2. Выберите «Совместный просмотр»
3. Отметьте аккаунты-участники и нажмите «Готово»

**Что фильтруется:**
- Watchlist (Хочу посмотреть)
- Up Next (Начать просмотр)
- История просмотров
- В процессе просмотра
- Коллекция
- Календарь

Бейдж на иконке меняется на `1+2` (номера участвующих аккаунтов). Для выключения совместного просмотра зайдите в меню и выберите «Выключить» или переключитесь на одиночный аккаунт.

### История просмотров
- Хронологический список всего просмотренного
- На постерах отображается дата просмотра и номер серии (S·E) для эпизодов
- Клик по карточке открывает полную информацию о фильме или сериале
- Долгое нажатие → удаление записи из истории на Trakt

### Статус просмотра сериала
На карточке и в детальном просмотре сериала отображается один из трёх статусов:
- **Смотрю** — есть прогресс просмотра
- **Завершено** — все вышедшие серии отмечены просмотренными
- **Брошено** — вручную помечен как заброшенный (сохраняется локально)

### Управление историей из карточки
Кнопка «Добавить в историю» на странице фильма или сериала позволяет:
- Отметить весь сериал просмотренным
- Отметить последний сезон
- Отметить последний эпизод
- Убрать из истории

### Watchlist и коллекция
Кнопки добавления/удаления из watchlist и коллекции Trakt прямо на странице фильма или сериала.

### Контекстное меню
Пункт «Списки Trakt» в контекстном меню любой карточки — быстрое добавление в пользовательский список.

### Мои списки
- Создание, переименование, изменение описания и удаление собственных списков
- Добавление и удаление элементов внутри списков

## Установка

1. В Lampa откройте **Настройки → Плагины → Добавить плагин вручную**
2. Вставьте URL файла `trakttv.js`
3. Перезагрузите Lampa
4. Откройте **Настройки → Trakt.TV** и нажмите **Войти**

### Получение Client ID и Client Secret

1. Перейдите на [trakt.tv/oauth/applications/new](https://trakt.tv/oauth/applications/new)
2. Создайте новое приложение
3. Укажите `urn:ietf:wg:oauth:2.0:oob` в поле Redirect URI
4. Скопируйте Client ID и Client Secret в настройки плагина

## Настройки

| Параметр | Описание |
|---|---|
| Client ID / Secret | Данные приложения Trakt |
| Отслеживать просмотр | Автоматически отмечать просмотренное при достижении порога |
| Порог прогресса | Минимальный процент просмотра для отметки (по умолчанию 80%) |
| Сортировка Watchlist | Порядок сортировки списка просмотра |
| Прогресс сериалов | Показывать статус просмотра на карточках сериалов |
| Синхронизация закладок | Режим синхронизации закладок Lampa с Trakt |

## Технические детали

- **Trakt API v2** — авторизация OAuth 2.0, все запросы через `Bearer` токен
- **TMDB** — обогащение карточек локализованными постерами и названиями делегируется в Lampa TMDB
- **Мульти-аккаунт** — слоты 0–N хранятся в `Lampa.Storage` как `trakt_slot_N_*`; активный слот задаётся через `trakt_active_slot`
- **Кеш совместного просмотра** — `_multiwatchIdsCache` хранит Set идентификаторов для каждого вторичного аккаунта; инвалидируется при смене аккаунта или изменении состава участников
- **CORS**: ряд Trakt-эндпоинтов (`GET /users/me/hidden/*`) недоступен из браузера — соответствующие данные хранятся локально
- Поддерживает Lampa 3.0+ (`Lampa.Maker`) и более старые версии (`InteractionCategory`)

## Изменения форка

- Убран счётчик лимитов API (Trakt не отдаёт заголовки `X-RateLimit-*` через CORS)
- Исправлено определение статуса «Завершено»: сравнивается количество вышедших и просмотренных серий
- Исправлено несоответствие статуса фильма между значком на постере и страницей карточки
- Добавлен раздел **История просмотров** с датами на постерах
- История поддерживает удаление записей через долгое нажатие
- Добавлена поддержка **нескольких аккаунтов** с быстрым переключением через иконку в верхней панели
- Добавлен режим **совместного просмотра**: фильтрация всех списков по пересечению watchlist-ов выбранных аккаунтов
