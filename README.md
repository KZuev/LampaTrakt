# LampaTrakt

Форк плагина [Trakt.tv для Lampa](https://lampame.github.io/main/trakttv.js) от [LampaME](https://lampame.github.io/main/). Оригинальный плагин создан командой LampaME — все авторские права на исходный код принадлежат им. Этот форк создан с уважением к оригинальной работе и доработан для удобной работы с историей просмотров, статусами сериалов, несколькими аккаунтами и совместным просмотром.

Дополнения и изменения, внесённые в этом форке, распространяются под лицензией [MIT](LICENSE).

Текущая версия: **3.0.22**

## Возможности

### Авторизация
Вход через OAuth 2.0 с собственными Client ID и Client Secret из приложения Trakt. После авторизации токены хранятся локально.

### Источник контента
Плагин регистрирует источник `trakttv`, который добавляет на главную страницу строки с подборками:
- **Трендовые** — самые обсуждаемые прямо сейчас
- **Ожидаемые** — самые антиципируемые новинки
- **Популярные** — лидеры по просмотрам за всё время
- **Рекомендации** — персональные рекомендации на основе вашей истории

Все карточки показываются с русскими названиями и постерами через TMDB. Разделы по жанрам (например, «Тренды: Боевик») отображаются на русском.

Поиск через источник TraktTV возвращает результаты на русском языке.

### Меню Trakt
Раздел в боковом меню Lampa с разделами:

| Раздел | Описание |
|---|---|
| Начать просмотр (Up Next) | Следующие непросмотренные серии сериалов из вашего watchlist |
| Список просмотра (Watchlist) | Все добавленные фильмы и сериалы |
| История просмотров | Хронологическая лента просмотренного контента |
| Расписание | Календарь выхода новых серий и цифровых релизов фильмов |
| Рекомендации | Персональные рекомендации Trakt |
| Мои списки | Управление пользовательскими списками |
| Понравившиеся списки | Списки, добавленные в избранное |

### Несколько аккаунтов

Плагин поддерживает одновременное хранение нескольких аккаунтов Trakt. Для быстрого переключения между ними в верхней панели Lampa появляется иконка с бейджем-номером активного аккаунта.

- Нажмите иконку → появится меню со списком аккаунтов
- Выберите аккаунт — плагин переключится и обновит все данные
- При выходе из аккаунта псевдоним (отображаемое имя) сбрасывается вместе с токенами

### Совместный просмотр (Multiwatch)

Режим, при котором все списки фильтруются по **пересечению** watchlist-ов выбранных аккаунтов: показываются только те фильмы и сериалы, которые добавлены у **всех** участников.

**Как включить:**
1. Нажмите иконку аккаунта в верхней панели
2. Выберите «Совместный просмотр»
3. Отметьте аккаунты-участники и нажмите «Готово»

**Что фильтруется:**
- Watchlist (Хочу посмотреть)
- Up Next (Начать просмотр)
- История просмотров
- В процессе просмотра
- Коллекция
- Календарь

Бейдж на иконке меняется на `1+2` (номера участвующих аккаунтов). Для выключения совместного просмотра зайдите в меню и выберите «Выключить» или переключитесь на одиночный аккаунт.

### Строка «Хочу посмотреть» на главной

Строка Watchlist на главном экране показывает только уже доступные фильмы и сериалы:
- Скрываются позиции с датой релиза в будущем
- Скрываются фильмы без известной даты выхода
- Скрываются фильмы с предстоящим цифровым релизом (они отображаются в разделе «Расписание»)

### История просмотров
- Хронологический список всего просмотренного
- На постерах отображается дата просмотра и номер серии (S·E) для эпизодов
- Клик по карточке открывает полную информацию о фильме или сериале
- Долгое нажатие → удаление записи из истории на Trakt

### Статус просмотра

На карточке сериала или фильма кликабельный статус показывает текущее состояние и дату последнего просмотра:
- **Просмотрено полностью** — все вышедшие серии отмечены просмотренными
- **Смотрю · S2 · E5** — есть прогресс, показан последний просмотренный эпизод
- **Просмотрено / Не просмотрено** — для фильмов

Рядом со статусом «Просмотрено» в скобках указывается дата (например, `15 мая 2025`). Если дата не известна или не записана — «неизвестно».

При нажатии на статус открывается меню:

**Для сериалов:**
| Пункт | Действие |
|---|---|
| Отметить как просмотрено полностью | Отмечает все сезоны и серии просмотренными |
| Отметить как не просмотрено | Удаляет весь сериал из истории |
| Хочу посмотреть / Не буду смотреть | Добавляет или убирает из Watchlist |

**Для фильмов:**
| Пункт | Действие |
|---|---|
| Отметить как просмотрено сейчас | Записывает просмотр с текущей датой |
| Отметить как просмотрено с неизвестной датой | Записывает просмотр без даты |
| Отметить как не просмотрено | Удаляет фильм из истории |
| Хочу посмотреть / Не буду смотреть | Добавляет или убирает из Watchlist |

### Управление историей из карточки
Кнопка «Добавить в историю» на странице фильма или сериала позволяет:
- Отметить весь сериал просмотренным
- Отметить последний сезон
- Отметить последний эпизод
- Убрать из истории

### Watchlist и коллекция
Кнопки добавления/удаления из Watchlist и коллекции Trakt прямо на странице фильма или сериала.

### Расписание
Раздел объединяет два типа контента:
- **Сериалы** — ближайшие эпизоды из вашего Watchlist (следующие 14 дней, по одной записи на сериал)
- **Фильмы** — фильмы из Watchlist с предстоящей датой цифрового релиза (тип 4 в TMDB)

Для эпизодов отображается скриншот кадра, для фильмов — постер или кадр из фильма.

### Синхронизация TorrServer ↔ Trakt

Двусторонняя тихая синхронизация прогресса просмотра без визуальных изменений:

- **TorrServer → Trakt**: при остановке до порога прогресс сохраняется через `POST /scrobble/pause`
- **Trakt → TorrServer**: при запуске Lampa или открытии карточки сериала прогресс из `/sync/playback` восстанавливается в `Lampa.Timeline` — TorrServer показывает прогресс-бар на позиции, записанной на другом устройстве

Работает только для эпизодов, которые хотя бы раз воспроизводились через TorrServer (плагин запоминает хэш файла → эпизод при каждом старте плеера и при открытии списка файлов торрента).

### Контекстное меню
Пункт «Списки Trakt» в контекстном меню любой карточки — быстрое добавление в пользовательский список.

### Мои списки
- Создание, переименование, изменение описания и удаление собственных списков
- Добавление и удаление элементов внутри списков

## Установка

1. В Lampa откройте **Настройки → Плагины → Добавить плагин вручную**
2. Вставьте URL файла `trakttv.js`
3. Перезагрузите Lampa
4. Откройте **Настройки → Trakt.TV** и нажмите **Войти**

### Получение Client ID и Client Secret

1. Перейдите на [trakt.tv/oauth/applications/new](https://trakt.tv/oauth/applications/new)
2. Создайте новое приложение
3. Укажите `urn:ietf:wg:oauth:2.0:oob` в поле Redirect URI
4. Скопируйте Client ID и Client Secret в настройки плагина

## Настройки

| Параметр | Описание |
|---|---|
| Client ID / Secret | Данные приложения Trakt |
| Отслеживать просмотр | Автоматически отмечать просмотренное при достижении порога |
| Порог прогресса | Минимальный процент просмотра для отметки (по умолчанию 90%) |
| Прогресс сериалов | Показывать статус просмотра на карточках сериалов |
| Синхронизация закладок | Режим синхронизации закладок Lampa с Trakt |
| Сброс всех настроек | Удаляет все токены, настройки и кэши плагина из localStorage |

## Технические детали

- **Trakt API v2** — авторизация OAuth 2.0, все запросы через `Bearer` токен
- **TMDB** — локализация названий и постеров: все ленты (тренды, поиск, расписание, watchlist) обогащаются через TMDB API на языке интерфейса Lampa
- **Мульти-аккаунт** — слоты хранятся в `Lampa.Storage` как массив `trakt_accounts`; активный слот задаётся через `trakt_active_slot`
- **Кэш совместного просмотра** — инвалидируется при смене аккаунта или изменении состава участников
- **Кэш предстоящих фильмов** — `trakt_upcoming_movie_ids` хранит TMDB ID фильмов с будущим цифровым релизом; обновляется при загрузке Расписания; используется фильтром строки Watchlist на главном экране
- **hashMetaCache** — Map `hash → {card, season, episode}` с TTL 7 дней; заполняется при старте плеера и при открытии списка файлов торрента (событие `torrent_file`)
- **CORS**: ряд Trakt-эндпоинтов (`GET /users/me/hidden/*`) недоступен из браузера — соответствующие данные хранятся локально
- Поддерживает Lampa 3.0+ (`Lampa.Maker`) и более старые версии (`InteractionCategory`)

## Изменения форка

- Исправлено определение статуса «Завершено»: сравнивается количество вышедших и просмотренных серий
- Исправлено несоответствие статуса фильма между значком на постере и страницей карточки
- Добавлен раздел **История просмотров** с датами и номерами серий на постерах
- История поддерживает удаление записей через долгое нажатие
- Добавлена поддержка **нескольких аккаунтов** с быстрым переключением через иконку в верхней панели
- Добавлен режим **совместного просмотра**: фильтрация всех списков по пересечению watchlist-ов выбранных аккаунтов
- Добавлена **двусторонняя синхронизация прогресса** TorrServer ↔ Trakt через `Lampa.Timeline`
- Строка Watchlist на главной скрывает недоступные и предстоящие фильмы
- Статус просмотра показывает дату в скобках (формат «15 мая 2025»)
- Все источники данных (тренды, поиск, расписание) локализованы через TMDB
- Меню управления статусом переименовано для ясности; пункт «В процессе» убран из меню сериала
- Сброс настроек полностью очищает все ключи плагина из localStorage
